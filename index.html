Comment installer

1

Téléchargement

Rendez-vous sur la page Releases du projet et téléchargez le fichier .msi le plus récent.

Ou cliquez ici pour le lien direct :

Chargement...
2

Installation

Lancez le fichier .msi téléchargé et suivez les instructions à l'écran pour installer l'application.

3

Message de protection Windows

Si Windows affiche un message "Windows a protégé votre ordinateur", cliquez sur Informations complémentaires puis sur Exécuter quand même.

4

Prêt à l'emploi

L'application est maintenant installée et prête à être utilisée. Les mises à jour futures seront proposées automatiquement.
